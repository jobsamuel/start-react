# start-react

> A great template to start developing with [React](http://facebook.github.io/react/) and [Material-UI](http://material-ui.com/).

# usage

### setup

	$ git clone git@github.com:jobsamuel/start-react.git
	$ npm install
	$ gulp

### build
	
	$ gulp build
